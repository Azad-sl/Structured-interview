# 四、应变处突

👇

<p class="tip"><BIG>定义</BIG></p>

设置一种或者多种情境，并给考生设定一个具体的身份，要求其能够站在题中所设定身份的立场和角度，对突然发生的各种状况、问题以及矛盾，做出反应并妥善解决。

- 身份设定上比较广泛，既有机关单位工作人员，也有公园、银行、社区、学校等的工作人员。



<big>常考类型：</big>

- **工作危机处理（工作应急）**
  - 条件缺失、舆情处理、误会疑问、争执纠纷、事故障碍、时间冲突
- **公共危机处理（突发公共事件）**
  - 人为事故、自然灾害
