# 一、综合分析(综合归纳+发散分析)

👇

<p class="tip"><big>常见考点</big> </p>

1. 针对**现象或问题提出自己的看法**
2. 针对**名言俗语、故事等谈理解、启示**
3. 对策建议类题目**针对问题谈解决问题的建议**
4. 观点类题目**针对不同观点谈看法**
5. 广东事业单位面试中还曾经出现过**观点式及漫画式**的考查形式



---

## 时政类(社会现象、公共政策)

- 话题：
  - 社会、文化、政治、经济

- 考点举例：
  - 中国动漫、思政课、广场舞、人工智能、公款吃喝、四风建设、农村电商、互联网+、公民素养、师德师风、网络安全、旅游乱象、山寨食品、人才大战、人情消费、公共服务、环境治理、扶贫乱象、资格证挂靠、垃圾分类、电子阅读……

### # 社会现象

<p class="warn"><b>积极类、消极类、辨证类</b></p>



💬 举例：

-  现在，很多人通过微博、微信等平台上发布的"微文章"来阅读，这已经成了当下最流行的阅读形式。对于"微阅读"，有人认为这是一种好的阅读方式，但也有人认为其存在很多问题。**对此，你怎么看？**



#### ❕ **答题思路**：

1. **扣题**

   - `背景铺垫` + 简单重复一下题干问题。
     - 社会背景、大政方针、身边例子、目的初衷
     - 点名现象本质 + 表态

2. **表态**

   - ##### **`主观表态`** ：

     - 积极：健康向上、乐见其成、鼓励发展
       - eg：<span style="color:blue">我对这种行为，持支持态度/    我非常赞同，我为他点赞/    我们应该要给予充分的肯定和点赞/    是一种值得赞赏的行为，我们应该向他学习/    给我们诸多的启发和教育/    的做法让我内心深受感动</span>
       - eg：<span style="color:blue">是当前我国政府职能从管理型政府到服务型政府、法制型政府转变的重要举措（有益尝试）/    这一问政于民的举措，值得肯定和提倡。这反映了政府全心全意为人民服务的理念，也是我们建设服务型政府的具体体现/    这正是我们政府和工作人员执政理念的转变，是我们政府执政方式的一种进步，更是我国提倡建立服务型政府的一种体现，所以值得大力提倡</span>
     - 消极：扭曲病态、令人心寒、无法认可
       - eg：<span style="color:blue">这必须引起我们的高度重视和反思/    需要引起我们高度的警觉/    对此我感到很惋惜/沉痛，应该引起我们关注和思考/    我认为......现象值得我们深思/    ......我认为缺乏了办事的温度</span>
     - 辩证：内因复杂，有好(利)有坏(弊)，不能一概而论
       - 我认为应该全面分析，理性看待/    需要我们客观、全面的审视/    要坚持具体问题具体分析；面对......现象，我们应该理性看待，认真思考，及时调整，这样才能避免许多问题/    这对于......来说有许多的积极意义，但同时也出现了一些问题隐患，需要我们全面客观地看待/    我认为应当对......给与正确引导，从而推动行业健康良性发展

3. **分析**

   - ##### 影响： **`直接——间接——深层次——短期——长期`** 

     - 积极：有益于，有利于，有助于，增加、提升、开启、推动、完善
       - eg：<span style="color:blue">青年人到基层和人民中去建功立业，其意义是不言而喻的：一方面可以充实基层的力量，推动党和国家各项事业的发展；另一方面可以锻炼青年的才干，磨砺青年的意志，为我们党和国家培养出一大批有基层实干经验和能力的青年干部。</span>
     - 消极：无益于、不利于、不仅无助于，还可能、减少、削弱、降低、阻碍
       - eg：<span style="color:blue">如果任由腐败滋长而不加以制止，将贻害无穷：一是侵害人民群众的合法权益；二是破坏机关干部队伍的建设工作；三是动摇我们党和国家的执政基础。</span>
     - 辩证：从积极的层面看，从消极的层面看

   - **问题：**

     - 当前存在问题，可能衍生问题

     - 现象如果是好的、是积极的，背后是否也能反映出一些问题

     - eg：<span style="color:blue">在“盲盒经济”迅猛发展的同时，也带来了一系列问题：第一，产品质量难以保障，三无产品时有出现：第二，消费纠纷难以解决，售后服务难以保障：第三，消费者非理性消费，可能造成资源浪费。</span>

   - ##### 原因：

     - **`内因、深层原因`**
     - **`外因、表面原因`**
     - eg：<span style="color:blue">关于贪污腐败原因的分析有很多，我认为最核心的几个点是：第一，领导干部理想信念不坚定；第二，公权力的运用缺乏有效约束；第三，法治建设不够完善。</span>

   - ##### # 肯定成绩：

     - 对相关方面为此做出的努力的肯定

4. **过渡**

   - <u>那么如何鼓励支持推动、预防消除遏制……该现象/行为？</u>

5. **对策**

   - 目前的对策（标），长远的对策（本）

   - **`强监管`** ：加强监管监督力度

   - **`重制度`** ：制定完善相关制度

   - **`提认识`** ：提高个人意识、社会整体意识；利用新闻媒体和社会集体宣传手段

   - eg：<span style="color:blue">想要从根本上杜绝以权某私的现象，真正做到权为民所用，需要重点在几个方面进行发力：第一，筑牢思想防线，使人“不想腐”；第二，完善约束机制，使人“不能腐”：第三，健全法律制度，使人“不敢腐”。</span>

6. **结尾扣题**

   - **`展望升华主题`**
   - eg：<span style="color:blue">我相信，通过加大宣传引导，强化治理力度，完善法律法规等措施，文明之花定会在农村处处绽放，从而进一步推动乡风文明建设，践行社会主义核心价值观，实施好乡村振兴战略。</span>
   - eg：<span style="color:blue">相信通过以上措施，一定可以实现短视频网络环境的风清气正，以看见的正能量帮助青少年扣好人生第一颗“扣子”。</span>


### # 公共政策

<p class="warn"><b>国家政策、地方政策</b></p>



💬 举例：

- 针对私家车长期占用公共停车位的现象，某市拟出台政策，对免费的公共停车位进行收费。这一政策在社会上引发了广泛热议，有人支持，有人反对，**你怎么看？**



#### ❕ **答题思路**：



<p class="tip"><code><BIG>国家政策</BIG></code> 都是好的、积极的、势在必行的。<code><BIG>地方政策</BIG></code> 则具体问题具体分析。</p>

1. 扣题
   - 重复一下题干问题，简单介绍政策背景。
2. 表态
   - 是否 **`合法合规`**
   - 是否 **`立足民生`**
   - 是否 **`势在必行`**
   - 是否 **`治标治本`**
   - 是否 **`兼顾到多方利益`**
3. 分析
   - 出台背景（ **``理论背景``+``社会背景``** ）+ 原因（ **``内因``+``外因``** ）
   - 影响/利弊（ **``正面``+`反面`** ）
   - 适用对象、针对的问题
   - 落实效果
   
4. 过渡
   - <u>如何使政策落实到位？如何更好地解决该政策所针对的问题？</u>
5. 对策
   - **`抓落实`**
   - **`提建议`** （完善，更好地兼顾各方利益）
6. 结尾扣题
   -  **`升华主题`**



## 哲理类(名言俗语、哲理故事、观点)

- 话题：
  - 名言俗语、寓言故事、领导人讲话

- 考点举例：
  - 新时代的青年人、青年价值观、好干部五条标准、为人处世……

### # 名言俗语

💬 举例：

- 在五四运动 95 周年之际，习近平总书记到北京大学考查，勉励青年学生"人生的扣子从一开始就要扣好"。**请联系实际，谈谈你对这句话的认识。**

### # 哲理故事

💬 举例：

-  一块冰在沙漠里被融化到很小，它抱怨说∶"要是在南极就好了，就不会融化了。"沙对它说："沙漠里缺少冰，你在沙漠里就显得非常珍贵，而南极的冰很多，你到那里就失去了价值。"**谈谈这个故事给你的启示。**

### # 观点

💬 举例：

- 有人说成功不能妥协，不妥协是成功的基石；也有人说成功需要妥协，妥协是一种智慧。**请结合自身经历，谈谈妥协和成功的关系。**



### ❕ **答题思路**：

1. 扣题
2. 表态
   - 启示值得深思
3. 分析
   - 启示/认识①：说理+举例
   - 启示/认识②：说理+举例
   - 启示/认识N：说理+举例……
   - 把握好 **`主要方面`** 与 **`次要方面`**
4. 过渡
   - 吸取经验教训，引以为戒、以此借鉴
5. 对策
   - **结合工作生活实际谈怎么做**
6. 结尾扣题
   -  **`升华主题`**





## & 态度观点

- 方向：
  - 单观点、双观点
- 考点举例：
  - 电动车出行、自行车道、沙漠污染、旅游
- 思路：
  -  **`提出观点`** ：表态，总体看法
  -  **`论证观点`** ：充分说明观点的合理性
  -  **`落实观点`** ：结合实际，如何将观点应用到工作、生活中
